# CoinQuest - Budget Tracker App

CoinQuest is a fully functional Android prototype for tracking personal expenses, setting monthly goals, and generating spending reports.

## Features
- **User Authentication**: Secure login and registration.
- **Category Management**: Create and organize spending categories.
- **Expense Entry**: Track expenses with date, time, description, and optional photo attachments.
- **Monthly Goals**: Set minimum and maximum spending targets using interactive SeekBars.
- **Detailed Reports**: View spending history and category-wise totals for custom date ranges.
- **Offline First**: All data is persisted locally using Room Database.

## Tech Stack
- **Language**: Kotlin
- **Database**: Room DB
- **Architecture**: MVVM (Model-View-ViewModel)
- **UI**: Material Components, ViewBinding, ConstraintLayout
